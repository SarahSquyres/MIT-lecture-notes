Lecture 2: Branching and Iteration

Branching

  • Conditional program
  • if < condition >
  • elif/else

While Loop

  • while condition is true, run code
  • otherwise run other code
  • unbounded number of iterations
  • can end early via break

      • Iterate through numbers in a sequence

    • n = 0, initialize variable/counter
    • for n in range 5
      • range creates a sequence of numbers starting from 0 going to 5-1
    • print (n)

For Loop

  • for < variable > in range (< some_num >)
  • run code
  • when you know the number of iterations
  • can end early with break
  • use a counter

Range

  • start, stop, step
  • default value are start = 0, step = 1 and optional loop until value is stop -1
    • mysum = 0
    • for i in range (7(start), 10(step))
    • mysum += i
    • print(mysum)

    • mysum = 0
    • for i in range (7(start), 10(step), 2(step, every other number))
    • mysum += i
    • print(mysum)
  • break statement ends loop
